typedef void(* VitureImuPoseCallback) (float *data, uint64_t timestamp)

Callback function type for imu pose data.

Parameters
dataPointer to pose data buffer
timestampTimestamp in device timebase for IMU sample.

Should be set before calling xr_device_provider_open_imu with VITURE_IMU_MODE_POSE Coordinate system: North-West-Up (NWU), X->North, Y->West, Z->Up. Data format: [roll, pitch, yaw, quaternion_w, quaternion_x, quaternion_y, quaternion_z]

◆ VitureImuRawCallback

typedef void(* VitureImuRawCallback) (float *data, uint64_t timestamp, uint64_t vsync)

Callback function type for imu raw data.

Parameters
dataPointer to raw data buffer (device-defined layout).
timestampTimestamp in device timebase for IMU sample.
vsyncVSync timestamp associated with the sample.

Should be set before calling xr_device_provider_open_imu with VITURE_IMU_MODE_RAW Data format:

  1. Viture One / Pro / Lite: [gyroscope_raw_x, gyroscope_raw_y, gyroscope_raw_z, accelerometer_raw_x, accelerometer_raw_y, accelerometer_raw_z, 0, 0, 0, temperature]
  2. Viture Luma / Luma Pro / Beast: [gyroscope_raw_x, gyroscope_raw_y, gyroscope_raw_z, accelerometer_raw_x, accelerometer_raw_y, accelerometer_raw_z, magnetometer_raw_x, magnetometer_raw_y, magnetometer_raw_z, temperature]
