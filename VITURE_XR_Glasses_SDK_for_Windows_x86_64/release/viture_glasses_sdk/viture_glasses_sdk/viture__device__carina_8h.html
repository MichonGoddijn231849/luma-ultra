Detailed Description

Carina-specific device API and callback types.

Copyright
2025 VITURE Inc. All rights reserved.

The Carina device (Viture Luma Ultra) provides higher-level callbacks for pose, IMU, VSync and camera frames. These callbacks are only applicable when the provider was created for XR_DEVICE_TYPE_VITURE_CARINA.

Get IMU pose data with prediction time (Twb matrix in OpenGL coordinate system: x -> right, y -> up, z -> backward)

Parameters
handleHandle to the XRDeviceProvider instance
poseArray to store pose data (7 floats for position and quaternion)
predict_timePrediction time in nanoseconds, 0 for current pose data
pose_statusStore pose status, 0 is good, 1 is bad. Can be null.
Returns
0 on success, -1 on failure
